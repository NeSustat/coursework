Тема курсовой работы (GUI1): 
    Написать утилиту командной строки для генерации изображения wordcloud на основе списка слов

Пояснение: 
    Программа должна принимать в качестве аргумента командной строки список слов и генерировать изображение wordcloud в виде файла jpg.. Необходимо программу на C++17, модульные тесты при помощи gtest.   В качестве библиотеки для работы должна использоваться библиотека qt (возможно использование другой библиотеки, например, SFML). Пример работы https://wordart.com/
